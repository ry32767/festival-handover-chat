# 04 食品模擬パート・1年企画パート 引継ぎナレッジ

対象年度: 第79回文化祭(2026年/令和8年)。参照比較: 第78回(2025/R7・テーマ「Crystal」)、第77回(2024/R6・テーマ「BIG BANG」)。第79回テーマは「SOLA」。
文化祭当日: 2026年は 5/9(土)校内祭・5/10(日)一般祭(1年企画スライドより)。2025年(R7)は 5/9(金)・5/10(土)。

## 1. 食品模擬パートの役割

- 【事実】文化祭スタッフ(文スタ)校内企画部に属し、**3年生の屋外食品模擬店**(食模擬)を統括する。第78回はチーフ・サブチーフの2名体制(役職名のみ記載)。
- 主な業務:
  - 食品メニュー候補(食品案)の作成・絞り込みと**保健所提出用の食品候補一覧**の作成
  - 3年の**文化祭委員会**(第1回〜第5回)の運営と各種書類の配布・回収
  - **衛生講習会**(店長対象・参加義務)・**試作会**の実施
  - 当日の模擬店管理体制(模擬本部)・雨天時対応の策定(配置移動パートと共同)
  - 前年1月に次期3年(現2年)へ**説明会+メニューアンケート**を実施
- 【事実】各クラスには「文化祭実行委員」(1学期始業式の日に決定。クラスとスタッフの仲介、書類準備、当日の見回り)と「店長」(1学期以降適宜決定。シフト作成、衛生講習会参加、模擬店運営責任者、在庫管理)を置く。
- 【事実】調理器具は**LPガス**による鉄板焼き機・フライヤー・コンロ・焼き鳥機・たこ焼き機・ピザ窯・綿あめ機・ワッフルメーカー等を使用。電子レンジ調理も候補にあり(焼きおにぎり)。
- 屋外食販/校外業者との関係: 【推測】LPガス調理機材はレンタル業者手配と思われるが、業者名・契約手順は今回の資料には**記載なし(不明)**。

## 3. 確定事実・ルール(数値・条件)

### 衛生・出店ルール(R7説明会スライドより)
- 感染対策が第一優先。アレルギー/衛生面に注意。
- **作り置きは避ける**。
- 調理に時間・手間がかかるものは避ける。調理方法/手順も含めて検討(焼く/炊く/煮る)。
- **揚げ物は大量の油の事後処理が大変なので1店舗くらい**に。
- 天候の影響が少なく、ゴミがあまり出ないメニューに。
- **食事もの5店舗・スイーツ系3店舗くらい**になるよう調整(=3年8クラスで8店舗)。
- **おにぎりを握って提供することは衛生上できない**(焼きおにぎり(冷凍・レンジ加熱)なら検討可。76thではモヤシ炒め+焼きおにぎりの組合せ例)。
- 各クラスでコンセプトを決める。

### 予算(【事実】R7説明会スライド「昨年の予算」)
- **1クラス 9万円 + パート予算 34万円**。

### 出店実績・候補

- 第79回(2026)食品案 ver.1 候補: 焼きそば / ポテト / から揚げ / クレープ / チュロス / 焼き鳥 / チーズボール / ラーメン / ハンバーガー / タコス / パンケーキ / スモア(焼きマシュマロ) / フランクフルト・ホットドック / チャーハン / 焼きおにぎり。
  - 検討事項: ラーメン(手間・汁の後処理)、ハンバーガー→ホットドック化、米もの(アンケートで米・おにぎり要望多いが握り提供不可)。
- 第79回(2026)食品案 ver.2 候補(ver.1から絞り込み): 焼きそば / ポテト / から揚げ / クレープ(コンロ加熱に変更) / チュロス / 焼き鳥 / チーズボール / **まぜそば**(ラーメンから変更・実施可否検討中) / タコス / スモア / フランクフルト・ホットドック / チャーハン。ラーメン・ハンバーガー・パンケーキ・焼きおにぎりは落選。
  - 「焼きそば〜チュロスは昨年も実施しアンケートでも人気」と明記。
- 加熱条件の基本形: 冷凍食材は「**冷凍状態のまま加熱調理**」(フライヤー/焼き鳥機等)、生食材は混ぜて加熱調理、盛り付けのみの品(クレープ等)は具材を解凍の上盛り付け。

### 雨天時対応(【事実】食模擬雨天時の対応・両年版)
- ① 弱い雨: 看板をテント下にしまう等の対応で通常営業。
- (R7版)野外ステージがピロティに移った場合: 模擬店配置を一部変更。西棟前ピロティ側の1店舗を南棟前へ移動しテント入替。南棟前・ピロティのイートイン撤去。**営業開始2時間前以降の降雨なら配置変更せず通常配置**。
- ② 強い雨で屋外販売が困難: **調理のみ屋外(テント)、販売・飲食は屋内**。
  - 101〜104教室: イートイン→販売(2クラスずつ)。105: 模擬本部→イートイン(本部は106へ統合)。
  - R7版では 201(生徒会/朝文研→イートイン化、朝文研は203のESSと合同展示・両者了承済み)、206(空き教室→イートイン)、308(謎解き待機教室→混雑時の予備イートイン)も使用。
  - 屋外イートイン撤去。イートインの椅子は視聴覚室へ、その他移動机椅子は物理教室へ。
  - 待機列スペース確保のため机椅子を減らす(廊下の列が教室模擬の列と干渉しないように)。

### その他

- 【事実】B-1グランプリ等の昨年度同様の企画は実施予定(実施方法変更の可能性あり)。

## 4. 1年企画パートの役割・企画内容・フロー

### 役割
- 【事実】1年生の文化祭参加企画(校内装飾+販売補助)を企画・運営する。1年生は「校内の装飾を担当(イートインスペース、校舎の窓)」。
- 【事実】各クラスの文化祭委員がクラス制作の取りまとめ・スタッフとの伝達役。連絡は文化祭委員のグループLINE経由。

### 第79回(2026)の企画内容
1. **イートインスペース装飾**: 2クラスで1つの教室(101〜104教室)を装飾。装飾は主に教室の前後、余力があれば側面(窓側)も可。テーマは揃えても別々でも可。
2. **サイン作成**(2026新規): 駅・商業施設の誘導サインのように行き先を示す案内を作成。教室名札の内側の空洞に紐を通してプラスチック段ボール/段ボールを吊り下げ掲示+窓に道案内ポスター。プラ段1枚で全クラス分作成可能と見積り。
   - 経緯: R8/1/7 総務パートが「一年生企画 企画書」で誘導サイン作成を提案(昨年度のステンドグラスは立ち見防止の意義を果たせなかったため)→ 2026/3/30 1年企画パートが「サイン作成について(案)」で具体化。
   - 企画書記載のメリット: 不慣れな来場者の誘導 / 窓を覆えば立ち見防止。問題点: 模擬店・展示部活の配置情報を1年に伝える必要 / 1年がすべき仕事か / 誤情報の確認が必要。
3. **物販(グッズ)販売スタッフ**: ジュース・文化祭グッズの販売。各クラス委員以外から**有志5人程度**(部登録までに決定)、1人1時間ほど(要確認)。

- ※資料「81回生第1回文化祭委員会.pptx」(第79回向け)にはメイン企画として**ステンドグラス制作**(黒画用紙+セロハン、校舎の窓に貼る)が記載されているが、R8/1/7企画書は「昨年度のステンドグラス」と過去形で言及しサインへの変更を提案しており、**どの年にどちらを実施したかは資料間で矛盾(要確認・不明)**。最終配布文書(80回生_一年企画_配布用文章)にはステンドグラスはなく、イートイン装飾+サイン+物販の3本。

### 予算・買い出しルール(【事実】)
- 予算: **1クラス4,000円**(イートイン装飾+サイン合わせて)。それ以外はクラス費から/持ち寄り。
- 買い出しは基本、文化祭委員が行う。担任に購入物の確認→費用4,000円を受領→購入。**必ず現金で購入・レシート必ず保管**・個人の物と同時に買わない(製作関係のみのレシートに)。守られない場合は個人負担の恐れ。委員以外に頼む場合もルールを伝える。文化祭後に担任または会計へレシートを渡す。

### 制作上の禁止・注意(【事実】)
- **風船は使用禁止**(文化祭途中に割れる恐れ)。
- **段ボールに直接ペン等で書き込まない**(リサイクル不可になるため)。装飾は貼り付け等で。
- **壁にガムテープを貼らない**(壁がはがれる)。
- 段ボールはある程度用意あり。個人的に店からもらう場合は店に確認を取る。
- 制作物の保管場所: 講堂4階(R7)/校舎内にスペース確保予定(R8)。

## 6. 今年(第79回)の変更点・反省・教訓

- **1年企画のメイン企画を変更**: ステンドグラス(面積が小さく立ち見防止の意義を果たせず)→**誘導サイン作成**へ。「装飾+実用目的」を狙う方針転換。
- **食品案の変遷**: ver.1→ver.2でラーメン・ハンバーガー・パンケーキ・焼きおにぎりを落とし、まぜそば・タコス・スモア・チャーハン等を追加検討。落選理由は手間(汁の後処理)・調理工程・回転率。
- **雨天対応の教訓**(R7→R8): R7版は野外ステージのピロティ移動連動・201/206/308の転用まで詳細化。R8版(食模擬雨天時の対応)は101〜105+201/202中心に簡素化。**「営業開始2時間前以降の降雨は配置変更しない」**という判断基準は引き継ぐ価値あり。
- **1年アンケートの教訓**: 準備期間不足が最大の不満。HR等の全体制作時間の確保、イートインの認知向上(案内看板=サインの継続強化)が課題。
- 説明会→アンケート→食品案という**前年1月からのサイクル**が確立している。1/10説明会・1/15アンケート締切(R7実績)を目安に。
