## 1. アンケートの仕組み

### 実施方法
- 生徒用・来場者用とも **Googleフォーム**。回答は文化祭直後(2025年は5/10〜、2026年は5/12〜)に収集。
- 来場者用はイートインスペース設置等で案内(R7反省点に「イートインに設置する場合は商業の授業も考慮」とあり)。回答の集中時刻(2026年は5/13朝7〜9時に大量回答)から、**学校連絡アプリ等で保護者へ一斉配信されたと推定**(配信手段の明記は資料になし=不明)。

## 3. 自由記述のテーマ別集約

### チケット・招待制度(最多テーマ。両年・生徒/保護者共通の不満)
- R7生徒: 「少なかった」206件。1,2年2枚・3年4枚は不平等 / 抽選結果発表が遅く招待の約束ができない / 出演部活のOB分・ステージ出演者優先枠が欲しい / 余ったチケットの返還・再配分制度が欲しい
- R8生徒: 改善案の自由記述で「枚数増」「電子チケット化」「家族はチケット不要に」「中学生以下は撤廃」「3年より1年に厚く(中学の友人が多い)」「OB枠新設」など。**2026年は「定員割れした」「客が少なく感じた」という指摘も複数** → 混雑対策と集客のバランスが論点。
- R8保護者(多数・強い要望): 「1人2枚では家族すら入れない」「家族枠を別に」「小学生以下はチケット不要に」「申込みそびれ救済を」「チケットをもらえず行けなかった」— **来場できなかった保護者からの回答が複数件**あり、次年度の最重要検討事項。

### 決済(au PAY・金券)
- 両年とも「**PayPayにしてほしい**」が頻出(R7生徒で8件以上、R8でも生徒・保護者双方で継続)。兵庫高校がPayPay導入済みとの言及あり。
- R7: 残額の使い道に困る / 返金が面倒 / 招待者にauPAYを入れてもらうのが大変。auPAY自体の操作性は92.6%が「使いやすい」。
- R8: 割り勘不可(本人確認未実施のため送金できない) / 入金できなかった / 回線混雑で決済に時間 / スマホ故障時に金券が助かった(金券併用の価値) / バラ金券の使い残し問題。
- R7会計反省: 金券は「完全な状態でないと換金不可」の周知不足、QRコード返却ルールの徹底、金券販売所の場所が分かりにくい。

### 食品模擬
- 称賛: 品揃え・味・低価格が好評(両年)。アレルギー表示(R8)が好評。
- 不満(R8で顕著): **売り切れが早すぎる**(12時台でほぼ完売、来場者から多数) / **支払列と受取列の区別が分かりにくい**(R8来場者で6件以上、「全店統一すべき」) / 待ち時間表示が実態と乖離(20分表示→実際1時間超が2件) / シフト交代時の販売一時停止。
- 衛生: R7「鶏肉が生焼け」「食中毒(疑い)が出た」、R8「唐揚げの中身が冷たい・一部生」「ポテトがべちゃっと」— **加熱不足の指摘が2年連続**。文スタによる調理チェック要望あり。
- 運営側の声: 試作会が1回では材料量の見極め不可 / 文化祭委員に情報が集中しクラスに共有されない / 前年の売れ行き・完売時刻の記録を残して発注量を決めたい(R8保護者提案)。

### 教室模擬
- 称賛: クオリティ高評価。**ファストパス・待ち時間表示・隣接教室を待機列にする運用が両年で好評**。待機中にCM動画を流す工夫(R8)も好評。
- 不満: 待ち時間長すぎ・回転率が悪い(巨大迷路は議事録段階から懸念されていた通り) / R8「20分表示で75分待ち、アナウンスなし、割込み放置」という強い苦情1件 / わたあめの煙・砂糖汚れ(R7)。
- 2年教室内食品模擬は両年とも約9割が継続支持。「前例がなく提供量が読めなかった」(R8)。

### ステージ(野外・講堂)
- 称賛: 野外バンドの盛り上がり、新ステージ(R7購入)、音楽部合唱・吹奏楽・演劇部・書道パフォーマンス、アイドルステージ(R8)。
- 不満: **野外と講堂の時間被り**(吹奏楽×ダンス部など、両年で複数件。第6回議事録でも懸念されたまま実施) / **講堂が暑い・満員で入れない**(R8で吹奏楽退場者多数、「体育館開催にしては」「2部制にしては」の提案、体調不良者発生の報告1件) / 撮影禁止の基準が不明瞭(R8) / 撮影カメラでドラムがボーカルに隠れて映らない(R8で2件、来年への具体的改善要望) / 講堂退場誘導スタッフの言葉遣いへの苦情1件(R8)。
- R7野外運営: バンド5分転換は無理 / 雨天ピロティ切替は的確だったが放送ブースのフル移動は無理 / シーバー不調時はLINE併用。

### WEBサイト(R8新設・Web開発班)
- **R8の目玉。生徒70件が「特に良かった」に言及**。混雑状況表示・ファストパス・長田検定ゲーム。保護者からも「事前情報で当日スムーズ」「クオリティに驚いた」と絶賛多数。
- 課題: 「WEBサイトが機能しないことがあった」(生徒1件) / FPの対象店舗が少ない・存在の宣伝不足 / 「AI製っぽい品質」という辛口1件 / ファストパス有料化提案(保護者1件)。

### ゴミ・清掃
- 称賛(R8で多数): **ゴミ分別・巡回回収が「ディズニーみたい」と絶賛**。他校文スタ経験者からも称賛。
- 不満(両年で継続): **ゴミ箱の場所が分からない/少ない**(R8来場者で10件以上)。パンフレットへのゴミ箱位置記載の提案(R8)。イートイン・各階・模擬店横への増設要望。

### 案内・導線・受付
- R7: 開場遅れと列対応、中学生受付の名簿照合で滞留、再入場者の突破、傘袋必須。
- R8: **「入場が昨年よりスムーズ」との称賛多数(5件以上)— R7反省が活きた成功例**。一方で校内地図が分かりにくい・最後尾が分からない・体育館横への行き方不明などの指摘は継続。「最後尾プラカード」は好評。私服の文スタが見分けられない→**識別できる服装/バッジの要望**(腕バッジは好評だった件も1件)。

### 日程
- **総体・公式戦との重複**が両年通しての最大級の構造問題(R7生徒で10件以上、R8保護者も複数)。「本人不在の文化祭は残念」「中間考査1週間前の開催に疑問」。
- R8保護者は「土曜開催はありがたい」(平日開催校が多い中)という支持も複数。

### その他企画
- 中学生企画: R7=スタンプ設置忘れ・景品場所不明。R8=クロスワードが難しすぎ・問題の場所が分からず**担当生徒が誰も把握していなかった**(専任配置の要望)。締切が早く中学生枠に間に合わなかった例。
- B-1: R7=auPAY投票は認知度不足。R8=LINE投票の合言葉「アストラ」が見つけにくい / 「従来のシール形式が良かった」(保護者)。
- フォトコンテスト(R8): 写真に番号がなく投票できなかった。
- 風船フィナーレ: R8は実施されなかった模様で「今年は風船がなく残念」(保護者)※実施可否の経緯は資料からは不明。
- PTA長田珈琲・制服リユースは両年とも好評。
- 自販機売切れ(水・お茶)が両年頻出 → 飲料の直販要望。

## 4. R7の反省点まとめ(反省点フォルダ・2025/5/24付PDF)

パート制の全パートが箇条書きで提出したもの。要点:

### 全体
- **文スタの負担・疲労が過大**(冒頭に明記)。一方で「仲がよく活動しやすかった」。
- チケット: 保護者に枚数が周知されずクレーム / 1,2年の枚数が少なすぎた。
- テーマ: 兵庫高校と大幅被り。開会式企画への反映しやすさも考慮を。
- 受付: ステージトップバッター(ダンス部)の保護者は別対応すべき / 決定事項は全受付に同時に下ろす / ボラスタのシフト引継ぎ失敗→事前説明会を / 傘袋必須。
- 本部: Discord・LINEの当日連絡が読み切れない→情報の交通整理を。

### パート別(来年の判断に効くもの)
- **実行委員長**: 前年副実長経験者が就くべき。HPは被リンクがないと検索に出ない→学校公式HPからリンクを。経路案内を実長1人でやらない。
- **副実長**: 2年を執行部員のみで編成しない。2日目昼にクラスシフトを入れない。ゴミ袋は早めに大量購入。偽学生証(中学生企画)の入金はGW前に必ず完了。
- **総務**: 開会式ウェーブ成功。B-1のauPAY投票は認知度不足→デジタル投票手段の再検討(→R8でLINE投票に変更済み)。詳細パンフは負担過大で存続要検討。氷・ジュースは学校着で発注(業スーから20kg引きずった)。物販は事務室の現金準備を待たず先生に立替依頼。
- **食品模擬**: 経験者必須。食中毒は「疑い」でも出してはいけなかった。床の養生テープは剥がれる→コーンに。
- **教室模擬**: 廊下混雑はR6より大幅改善(人数減+列並べ教室+チラシ)。わたあめは2時間ごと10分清掃で営業停止が発生。
- **1年企画**: パート3人は多い(1〜2人で可)、総務と合併したい。学年団との連携不足。
- **展示**: 2階活用は成功。物理部が無断展示でケガ人が出る恐れ→処遇検討。
- **野外**: 5人必要。購入ステージは正解(ただし濡らし厳禁、GW中に建てる必要はない)。雨予報なら最初からピロティ実施に。A3ラミネートをA4フィルム2枚で作ると濡れて破れる。
- **講堂**: 4人は多い(照明込みで3人がベスト)。
- **デザイン**: 4人がベスト。ロゴはCMYKで作成。入稿は塗り足し+3mm。総務の入金遅れでグッズ到着が遅れた→パート間連携。
- **装飾**: 幕のロープ数不足でヒヤリ。幕を解体しない場合は講堂2階へ(鍵は事務室)。
- **配置移動**: 2階利用は手間よりリターン大。3年生の備品無断持出しが不足トラブルの元→再発防止必須。復元移動は見込みが甘くトラブル多発、中庭に放送は聞こえない。
- **会計**: 2人でも可。部活予算の申請方式に不公平感→「最低限のみ記入+年度中追加申請」等の改善案。模擬店の値段設定管理を会計が持つ案。
- **備品**: 「貸せない物の箱」を作る。トランシーバーはチャンネルロック。小会議室の机がなくなったらまず講堂2階を探す。

### 反省点docx(2026/5/21付)= R8用ファイルだが**中身はほぼR7のまま**(ファイル名に「これを変更(昨年のをそのままにしています)」)。R8で追記済みの差分は以下のみ:
- 傘袋→「生徒指導部」から「**総務部**との連携」に修正
- 風船の破裂音が片付け時に鳴っていた→回収人員増、公演中は禁止/注意
- 足湯の電気をつける
- トランシーバーを本部に必ず1つ置く
- ※**R8の本格的な反省点は未記入 = 未完了タスク**
