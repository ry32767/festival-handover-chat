## 1. 役割と全体運営体制

### 実行委員長の役割(資料から読み取れる業務)
- 文スタのパート編成・名簿確定、テーマ決め取りまとめ
- 招待制度全体の設計(枚数・スケジュール・チケット製作指示)、先生方への配信依頼文書の作成
- 食べ歩きルール等の全校ルール提案(副実長・食品模擬と協議)
- ボランティアスタッフ募集の企画
- 受付・本部・運営マニュアルの作成、当日シフト作成
- 事後アンケートの実施・集計
- 開会宣言・閉会宣言・閉会式挨拶(開閉会式の司会は副実長)

### 副実行委員長の役割
- 実長のサポート・他パートのサポート(中学生冊子より「実長の右腕」)
- **ごみ処理の一切**(明記: 「文化祭におけるゴミ処理の一切は、副実行委員長の管轄」)+管理委員会の統括
- **中学生企画**の企画・製作・運営
- **SNS宣伝**(Instagram / X / 公式LINE)の運用と各パートからの宣伝依頼受付
- 開会式・閉会式の司会、生徒誘導

### 文化祭スタッフ(文スタ)の構成
- **2026年(第79回)**: 11パート+web開発班1班、計47名(2025/12/24付で確定。※名簿の列挙は46名分で1名分の不整合あり — 要確認)
  - パート: 副実行委員長(4名: チーフ1・副チーフ1含む) / 総務・一年(5) / 食品模擬(5) / 教室模擬(4) / 展示企画(4) / 野外(6) / 講堂(4) / デザイン(3) / 装飾(3) / 配置移動(2) / 会計(3) / web開発班(3)
- **2025年(第78回)**: 12パート、計42名(2024/12/17付)。「総務」と「1年企画」が別パート、web開発班なし
  - → 2026年の変更点: 総務+1年企画→「総務・一年」に統合、**web開発班を新設**(文化祭HP/WEBサイトはアンケートでも好評)
- 各パートにチーフ・副チーフを置く。実行委員長はパート横断で統括
- クラス側組織: **文化祭委員**(各クラス2名、クラス取りまとめ・情報伝達)、**管理委員**(各クラス2名、ごみ処理担当)、**会計委員**(各クラス2名)。4月上旬のHRで担任が決定(R7は4/8決定、4/8・4/10に委員会開催)

### テーマ
- 2026年: **SOLA**(「見上げる空に」「去年よりも高く、自由な」等のコピーで使用)
- 2025年: **Crystal**(候補6案: Crystal / UZUMAKI / Contrail / Up Grade / ANTENNA / 長田万博(Expo) から、12月のスタッフミーティングで6案に絞り→第1回合同会議(12/23)で先生の意向も踏まえ決定。副題はテーマ決定後チーフ会で決定)
- テーマ決めの時期: 12月中旬のスタッフミーティング(パート確定と同時期)

## 2. 主要業務の詳細

### 2.1 文スタ募集・編成
- 12月中旬〜下旬にパート編成を確定し、先生に名簿を提出(2025年版: 12/17、2026年版: 12/24)
- 名簿の書式: パート | 役職(チーフ/副チーフ) | 氏名 | 学籍番号 | 所属部活

### 2.2 招待制度(数値は確定事実)

#### 招待枚数(2026年案 = 2025年実績からの変更込み)
| 対象 | 2025(第78回) | 2026(第79回) |
|---|---|---|
| 1年生 | 各2枚(計640枚) | 各2枚(計640枚) |
| 2年生 | 各2枚(計640枚) | 各2枚(計640枚) |
| 3年生 | 各4枚(計1280枚) | 各4枚(計1280枚) |
| 在校生配布計 | 2560枚 | 2560枚 |
| 中学生(第1学区・保護者不可) | 上限300枚(先着/HP募集) | **350枚に増** |
| 確定再配布 | 140枚 | **190枚に増** |
| **来場者数全体の上限** | **3000人** | **3100人** |

- 2025年実績: 初回調査で1・2年627件=1223枚、3年309件=1141枚、計2364枚配布。再配布可能336枚に対し追加要求713枚(382件)。再配布可能率は人ベース88%・枚数ベース47%(=追加希望は満たしきれない)
- 追加チケットは**1人2枚まで**、枚数の少ない下級生(2026年は80・81回生)を優先
- チケットは連番(製番)入りで印刷。2026年版は**0001〜2750**の通し番号を印刷(8枚/ページ×344ページのPDF)
- チケット運用: **再入場可能・受付で回収しない**。招待「する」生徒の名前+学籍番号をチケットに記入してから渡す(招待される側の名前ではない)。配布時は担任へ封筒+整理番号入り名簿を添付。不備は当日17時までに生徒会室へ
- デザイン: 2026年版は搭乗券風(NSF2026 / INVITATION PASS / Date: 9 May 2026 / Admission 09:00 / 9:00〜15:00)。auPAYダウンロード案内を記載

- 配信は**学年副主任の先生に依頼**する形式(依頼文書を毎回作成)。文化祭委員にもHR連絡を依頼
- Google Form利用。教室掲示(QRコード)を併用。掲示は各クラスの日番BOXに投入

#### 中学生招待
- 対象: 第1学区中学校在籍の中学生のみ(**保護者・一般は不可**)
- 本校HPで告知・先着順、上限到達で締切。到達しない場合4/25(金)20時締切
- 2025年は電子チケットサービス「**Flippa(フリッパ)**」を利用(教育関連イベントは無料と確認済み)。QRは読み取らず**画面確認のみ**
- 当日は**申込完了画面(スマホor印刷)+生徒手帳(学生証)**を持参→受付で整理番号をもとに名簿にマーカー
- auPAY導入の案内も告知に含める(金券も利用可)

#### 先輩(前年度文スタ)招待
- 全校生徒・中学生配布分とは**別枠**
- 2026年: 第78回文スタを務めた78回生、最大21名 / 2025年: 77回生、最大23名
- Google Formで来場希望を調査→昨年スタッフだった先輩経由で紙チケット配布

#### 優先招待チケット(2026年新設・3/29提案)
- 目的: 開場直後の入場行列で、開会直後の舞台出演部活(2026年は**書道部**)のOB・OG・保護者がプログラムに間に合わない問題の解消
- 一般と識別できる色・デザイン、入場ゲートに**優先専用レーン**を設置
- 配布: 対象部活の生徒に直接説明・枚数調査(一般+優先合わせて各学年上限まで)
- 受付運用: **受付開始から10分間、9番窓口を優先チケット専用**→その後中学生窓口に転用

### 2.3 食べ歩きルール(2025・2026同一内容で継続)
- 食品模擬店の**待機列での飲食は可能**
- **校舎外での食べ歩きは原則不可**。ただし通路を塞ぐ・迷惑になっていない限りスタッフからの注意は行わない
- **校舎内での食べ歩きは禁止**(注意喚起を行う)
- 課題認識: スタッフ間で注意基準に差が出る→注意喚起マニュアルを作成する(と両年とも記載)

### 2.4 ボランティアスタッフ(ボラスタ)
- 募集開始: 2026年は3/20(金)〜(Classroom配信文では3/23(月)〜)。**締切は設けない**(多く集めるため)
- 仕事内容: 幕塗り、受付、列整備(パトロール)、ステージ見回り など。シフト制・当日のみの仕事も多い
- 募集方法: 修了式・部活動紹介での宣伝、教室掲示、SNS宣伝
- 応募方法: LINEオープンチャット「2026文化祭ボラスタ」参加(参加コードは募集時に別途配布)。名前は『学籍番号 名前(ボラスタ)』表記に統一。生徒会室に来る/文スタに声かけ/SNSのDMでも可
- 端末制限等でオプチャに入れない生徒向けに**連絡用公式LINEアカウント**を用意(同内容を発信、原則はオプチャ)
- 文スタ側: 連絡はオプチャ、専門的な仕事はトークルームを新設
- ごみ処理ボラスタにはフローチャートを配布(シフト前日までに確認→当日はシフト5分前に会議室で説明→分別/呼びかけ/運搬要請→閉会式後の最終分別)

### 2.5 受付・本部マニュアル

#### 入場受付(2026年版)
- 入場経路: ①入口=正門横の通用門(スタッフ誘導) ②待機場所=生徒会室横庭園・陸橋下、南門地点で折返し ③受付=図書室の真下・足湯横(受付+パンフ配布)
- **受付は9箇所**(2025年は5箇所+中学生用6番)。色分け: 赤=優先招待用、緑=中学生用(4・5・6番、混雑時以外は一般対応可)、青=パンフ置き場。開始10分間は9番=優先チケット専用→以後中学生窓口
- 一般来場者: チケット確認(回収しない・再入場可)→パンフレット案内(1人1部)→次の人を呼ぶ。パンフ切れ時はQRコードでHP案内
- チケット無しは**安全面の観点から入場不可**。例外は在校生保護者のみ: 子の学年・クラス・名前確認→生徒に電話→文スタが本人確認できれば入場可。連絡が取れなければ本部経由で試み、ダメなら入場お断り
- 中学生: 専用窓口へ誘導→申込完了画面確認→学生証確認→名簿にマーカー(2026年は本人に引いてもらう)→パンフ+中学生冊子(2026年はクロスワード入り、2025年はスタンプラリー入り)配布。**保護者同行は入場不可**
- ボラスタ心得: 答えられない質問は文スタに引き継ぐ。クレーム・手順に従わない人・高圧的な態度は文スタ対応(入場を待ってもらい速やかに報告)
- 2025年はパンフと一緒に「長田学園新聞」も配布(無くなり次第終了)

#### 受付統括(文スタのリーダー役)
- 仕事: 入場者数カウント(カウンター使用)、受付への案内、受付列整備・駐輪場誘導、トラブル対応、本部への連絡、受付補助
- 9:20から書道部(講堂)ステージがあるため受付後の経路アナウンスを実施(2025年は9:15から書道部+ダンス部)
- 呼びかけ例: 「1歩ずつ前にお詰め下さい」「au PAYアプリのダウンロードはお済みでしょうか」
- 自転車来場者は指定駐輪エリアへ(校外の塾の看板前に停めさせない)
- 受付統括シフト(2026年): 9:00〜15:00を1時間区切りで各パートチーフ・副実長が担当。9:00〜10:00が列整備の山場、14:00〜15:00担当は混雑状況に応じて全体運営へ

#### 本部
- 仕事: 連絡管理(**Discord・LINE・シーバー**)、質問対応、落とし物管理、auPAY返金対応、運営スタッフへの指示出し、トラブル記録
- 本部シフト: 2名×1時間交代。1日目9:30〜15:00、2日目9:00〜15:00
- 落とし物: 記録表(番号/拾得時間/品名/場所/色/特徴/受付サイン/返却サイン)に記入→番号を貼ってボックスへ。返却時は**ボックスの中身は見せず**特徴を先に聞く。スマホ=パスワード解除で本人確認、財布=持ち主しか知らない特徴を言ってもらう
- auPAY返金: 本部に先生がいれば先生へ、いなければ金券販売所(会議室)へ案内(本部を空にしない)
- 再発しそうなトラブルは記録して全体共有

#### 運営体制・シーバー(トランシーバー)
- 体制: 本部(全体統括・指示出し・忘れ物)/各部署(模擬店・ステージ)/受付統括(一般祭のみ)/全体(校内巡回・応援要請対応)
- シーバー配置 2026年: **計15台**(本部1、受付2、全体1、先生2、食品模擬1、教室模擬1、野外1、講堂1、実長1、副実長1、校内スタッフ2、予備1)。2025年は計13台(実長・副実長・予備なし、校内スタッフ4、先生1)
- 通話プロトコル: 「①パート(居場所)+名前です ②要件(トラブル/応援要請/○○お願いします) ③詳細」。**チャンネルは1に設定**、充電満タン確認

### 2.6 全体シフト
- 1日目(校内祭)9:30〜15:00、2日目(一般祭)9:00〜15:00を約1時間区切りで、教室模擬/食品模擬/講堂/野外/本部×2/全体×2/受付 の枠に文スタ全員を割り当て(2026運営についてに実シフト表あり。表記が愛称混じりでOCR乱れあり)
- 文スタの優先的な飲食(R7): **校内祭のみ**、文スタ名札着用者は食品模擬が混雑している場合に優先提供を受けられる(余裕がある時は一般と同じ扱い。待機列の邪魔にならない位置から受付に声かけ)。3年文化祭委員経由でクラスに伝達

### 2.7 事後アンケート(2026年実施・回答は5/12〜)

#### 生徒向け(回答233件: 1年121・2年50・3年62)
設問: ①学年 ②見たもの(複数) ③特に良かったもの ④困ったこと ⑤**2年生の食品模擬店はどうだったか** ⑥チケットの枚数 ⑦チケット抽選制度 ⑧(改善が必要と答えた人へ)改善方法 ⑨その他感想

主要結果:
- 2年生食品模擬: **継続希望220 / 改良必要13** → 圧倒的に継続支持
- チケット枚数: 抽選含めて丁度良かった112 / 抽選含めず丁度良かった31 / **少なかった88**(約38%)
- 抽選制度: 継続199 / 改善必要34
- 改善要望の傾向: 枚数増・開催日数増、電子チケット化、抽選結果の分かりやすい表示、家族枠と友人枠の分離
- 自由記述の頻出: **auPAYではなくPayPay等にしてほしい**(多数)、ファストパス好評、WEBサイト好評、中間考査と日程が近い、講堂と野外のステージ時間被り、模擬店売り切れ情報がInstagram頼み、中学生定員割れの指摘、OB・OG入場希望

#### 来場者(保護者)向け(回答390件)
設問: ①招待者との間柄 ②見たもの ③特に良かったもの ④困ったこと ⑤感想
- 特に良かったもの上位: **講堂ステージ118 / 野外ステージ83 / 食品模擬80 / 教室模擬62** / 部活動展示14 / 装飾10 / WEBサイト10
- 困ったことの頻出: 食品の早期売り切れ / **待ち時間表示と実際の乖離**(20分表示→実際1時間以上、お化け屋敷で1時間15分) / ゴミ箱の場所が分かりにくい / 支払列と受取列の導線が分かりにくい / チケット枚数不足(家族全員で来られない、祖父母を呼べない、家族枠要望) / 座る場所・休憩場所不足 / パンフ地図が分かりにくい / 講堂(吹奏楽)が満員で入れない・暑い / 自販機の水・お茶売り切れ / auPAYへの不満 / 文スタが私服で誰に聞けばいいか分からない
- 好意的意見: 生徒主体の運営・接客への称賛多数、金券がスマホ非所持layer(祖父母・弟妹)に有効
- 回答期限の掲示(R7): 所要5分程度・回答期限は一般祭当日17時

### 2.8 ごみ処理(副実長管轄)
- 収集形態: **ゴミステーション**(全ゴミ一括回収)を約5か所設置(R6実績: イートイン前廊下、1年教室前中庭ベンチ近く、ピロティ中庭側、体育館側、食堂前)。設置数は配置移動パートの全体配置決定後に確定
- ステーション製作: GW頃完成目標。材料/1か所あたり: 段ボール4箱(総務部の先生が確保)、ゴミ袋4〜5枚(文スタ所有)、トング1(総務部の先生)、机1(会議室)、手袋(保健室のビニール手袋)
- **管理委員**(各クラス2名)が当日ゴミステーション前で管理。任務時間は各1時間程度。任務: ①呼びかけ ②分別(手袋・トングで) ③運搬要請(袋が一杯→管理委員LINEグループへ連絡→文スタが回収)
- シフト: Google Formで第3希望まで、**4/22締切**。未提出者は残り枠に割り当てる場合あり。当日は開始5分前に生徒会室前集合で説明
- 2日目閉会式後、**プール横のゴミ本部**で全員最終分別

## 6. 今年(2026)の変更点・反省・教訓

### 変更点(2025→2026)
1. **来場上限 3000→3100人**、中学生300→350枚、再配布140→190枚に拡大。根拠: 混雑は収容人数でなく「人の流れの設計」の問題→**混雑状況の可視化**(空いているエリアを来場者が把握できる仕組み)+中学生企画による動線分散で対応。予算安定化と広報(志望者数減への対策)がメリットと整理
2. **優先招待チケット新設**(開会直後出演部活OB・OG向け、専用レーン)
3. **受付を5→9箇所に増設**、色分け運用導入
4. パート再編: 総務+1年企画統合、**web開発班新設**。人数42→47名
5. シーバー13→15台(実長・副実長・予備を追加)
6. 中学生企画をスタンプラリー1本→**4企画**(冊子・ツアー・謎解き・景品)に拡大、定員600名を明示
7. 中学生冊子の景品を外注ポストカード→内製の文スタカード等に変更
8. 2025年に配布していた長田学園新聞の配布記述が2026年マニュアルからは消えている(要確認)

### アンケートから得られた反省・教訓
- 決済: auPAYへの不満が生徒・保護者双方で最多クラス。**PayPay等への変更検討が最大の宿題**
- チケット: 「少なかった」38%。家族枠(保護者は枠外に等)・電子チケット化・抽選結果の可視化の要望
- 運営: 待ち時間表示の精度(20分表示→実際1時間超)、支払列/受取列の導線表示、ゴミ箱位置の案内(パンフ記載要望)、座席・休憩場所、パンフ地図の改善、文スタの識別性(Tシャツ等で分かるように)
- ステージ: 講堂(吹奏楽)の収容・暑さ、講堂と野外の時間被り、野外の日陰
- 中学生招待は定員割れ→告知強化 or 制度見直しの余地
- ファストパス・WEBサイト・金券(非スマホ層向け)は好評→継続
