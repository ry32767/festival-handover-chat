# 用語集 (glossary)

長田高校文化祭の運営資料に登場する用語・略語・固有名詞。五十音・カテゴリ順。

## 組織・人

| 用語 | 意味 |
|---|---|
| 文スタ | 文化祭スタッフ。生徒の運営組織(約40〜47名)。前年秋〜冬に募集 |
| ボラスタ | ボランティアスタッフ。当日の受付・警備等を手伝う生徒。LINEオープンチャットで運用 |
| 実長・副実長 | 実行委員長・副実行委員長。11月頃決定 |
| チーフ | 各パートのリーダー(◎)。副リーダーは○ |
| チサブ | チーフ・サブ(副リーダー)の総称 |
| 文化祭委員 | 各クラス2名の窓口委員。R6に「実行委員」から改称 |
| 管理委員 | 各クラス2名のゴミ担当委員(当日シフト各1時間) |
| 会計委員 | 各クラス2名の会計担当委員 |
| 店長・副店長 | 食品模擬店のクラス側責任者。衛生講習会の参加義務あり |
| 回生 | 学年の呼び方。卒業期数で呼ぶ(例: 80回生)。2026年時点で78回生=3年、79回生=2年、80回生=1年 |
| 運営委員 | 教員側の担当(各学年2名+専門部1名)。異動時の引継ぎルールが明文化されている |
| 合同会議 | 全パートチーフ+担当教員の意思決定会議。1〜5月に月2回(年8〜12回) |
| 一年生会議 | 1年生向けの説明・連絡会議 |
| 幹事 | 点呼担当 |

## 行事・企画

| 用語 | 意味 |
|---|---|
| 校内祭 / 一般祭 | 1日目(生徒のみ) / 2日目(招待来場者あり) |
| テーマ | 各回の祭名。HELM(76回)→BIG BANG(77回)→Crystal(78回)→SOLA 〜この空、長田色。〜(79回) |
| B-1グランプリ(B1) | 模擬店人気投票。閉会式で結果発表。シール投票の不正→第79回から公式LINE+合言葉方式。「N1」への改称案あり |
| 合言葉「アストラ」 | 第79回B1投票の合言葉。「隠れアストラ」「文スタカード」は関連の遊び企画 |
| 開会宣言リレー | 第79回開会式企画 |
| 野外フィナーレ | 閉会式後の締め企画。風船(77回)→カーテンコール(79回) |
| 足湯 | 1階の文スタ展示会場周辺の企画(総務/展示の管轄が曖昧) |
| 学塔企画 | 商業棟南側4〜6階を開放する眺望企画(77回。抽選チケット制) |
| 長田検定 | webサイト上のクイズ企画(79回) |
| ファストパス | 人気模擬店の優先入場券。webサイトと連動 |
| 中学生企画 | 中学生招待者向け企画。79回は冊子・ツアー・謎解き・景品の4本立て(定員600) |
| 食べ歩き | 校内での食品持ち歩き。ルール・注意基準は実長管轄 |
| 文化祭の友 | 全職員・文スタ用の当日運営マニュアル冊子(来場者パンフとは別物)。副実長が編集、4月下旬発刊 |
| 詳細パンフレット | 来場者向けの企画詳細冊子(総務) |
| CM | 各模擬店の宣伝動画。前日・当日に上映。79回に復活(15秒) |

## 模擬店・出店

| 用語 | 意味 |
|---|---|
| 教室模擬(教モギ・室模擬) | 2年生の教室出店(お化け屋敷・縁日・カジノ・迷路・脱出等) |
| 食品模擬(食模擬) | 3年生の屋外食品販売(テント、食事5+スイーツ3店舗) |
| 食A/食B | 食品系教室模擬の区分呼称 |
| 仮企画書 / 本企画書 | クラス提出の2段階企画書(4月中旬〆) |
| 予想収益計算表 | 待ち行列理論ベースの収益計画表(79回導入) |
| 必要物品表 / シフト表 / 食品表 | クラス提出の運営計画様式 |
| 宣誓書 | 運営ルール順守の誓約書(春休み明け配布) |
| 回転率 | 客1人あたり滞在の目安(最大10分・理想5分)。教室100人・廊下20人が定数 |
| 臨時営業 | 神戸市の食品営業届出(開催10日前まで。調理不可=市販品の小分け・トッピングのみ) |
| 検食 | 提供食品の保存サンプル(1品50g×2週間冷凍、当日朝プール横で提出) |
| 衛生講習会 | 店長対象の義務講習(4月下旬) |
| 試作会 | 食品模擬の事前調理練習(4/30頃) |
| イートインスペース | 飲食用教室(101〜104)。装飾は1年企画が担当 |
| 模擬本部 | 食品模擬の運営本部教室(105〜106。冷蔵庫・機材保管) |
| フォグマシン | お化け屋敷用煙発生装置。79回に教室内解禁(水性液・暗幕に当てない) |
| プラダン | プラスチック段ボール(黒。加工不可の貸出材) |

## 金・決済

| 用語 | 意味 |
|---|---|
| au PAY | 主決済(KDDI)。QRコードは翌年再利用(進路指導室保管)。手数料約2.6% |
| 金券 | 補助決済。現行20円単位、当日・当該回のみ有効、未使用のみ100円単位で払い戻し。台紙で管理 |
| 準備金(貸付金) | クラス・団体への事前貸付。余りは返金、超過は自腹 |
| 赤字補填 | 赤字の予算穴埋め。**第79回から全面廃止** |
| 特別会計 | 黒字の積立。長く使える備品(ステージ板・シーバー等)に充当 |
| 単価個数制度 | 物販の仕入・価格管理方式(総務) |
| 予約販売 | 物販の事前予約分(別置き管理) |

| 暗幕室 | 5階の幕・暗幕保管室。鍵=運営委員長の先生 |
| 外部倉庫 | 体育館下の倉庫。鍵=体育教官室 |
