# 長田高校文化祭 運営ナレッジベース (LLM用引継ぎ書 本体)

## 0. この文書の目的と使い方

この文書は、兵庫県立長田高等学校 文化祭の運営に関する全資料(元資料868ファイル)を、次年度担当者を支援するLLMが読み込むために統合したナレッジベースの**中枢(ハブ)**である。

- **この`master.md`だけで**: 全体像・体制・年間スケジュール・共通ルール・変更点・教訓に答えられる。
- **パート個別の詳細**(締切、価格、手順、業者、反省の全文)は `details/` の該当ファイルを読むこと。各パートのサマリ末尾に参照先を明記している。
- `glossary.md`=用語集 / `unresolved_issues.md`=未解決課題 / `source_index.md`=全ファイル索引 / `folder_tree.md`=元資料フォルダ構成。
- 元資料は `OneDrive_2_2026-7-10/` 配下。画像・動画・一部の画像PDFはテキスト化されていないため、資料カタログで「原本参照」とあるものは元ファイルを開く必要がある。
- 記載内容は2026年7月時点の資料に基づく。**推測は「推定」、資料にない事項は「不明」と明記**してある。

## 2. 運営体制

```text
実行委員長(実長)・副実行委員長(副実長)  ※11月頃決定、12月引継ぎ式
│
├─ 校内企画部: 教室模擬(2年) / 食品模擬(3年) / 総務 / 展示企画
├─ ステージ企画部: 野外 / 講堂
├─ コンセプト部: デザイン / 装飾   ※担当教員が同一で一体運用
├─ 配置移動 / 会計 / 1年企画(総務と併合・独立を年により繰り返す)
└─ web開発班(2026年新設)
```

- 各パートに◎チーフ(リーダー)・○副リーダー。パート横断の意思決定は**合同会議**(月2回、チーフ+担当教員。「例年やっていないことは必ず会議を通す」「会議未承認事項を生徒におろさない」が鉄則)。
- クラス側の窓口: **文化祭委員**(各クラス2名。R6に「実行委員」から改称)+会計委員・管理委員(ゴミ)各2名、食品店は店長・副店長。
- 教員側: 運営委員は各学年2人+専門部1人。分掌=総務部(保険・来賓・消防渉外)/生徒指導部(警備)/保健部(衛生・保健所渉外・救護)/教育企画部(記録)。校内異動時は新旧2人体制、転勤時は引継ぎ資料必須(明文規定)。
- 実長側の全体業務: 文スタ募集(前年秋〜冬)、招待制度、ボラスタ(LINEオープンチャット)、受付・本部、全体シフト、事後アンケート。副実長: ごみ処理、中学生企画、SNS、開閉会式司会、Webサイト(班新設前)。

## 4. 全体共通ルール(全パート共通の確定事項)

- **会計原則**: 立替禁止・レシート/領収書のみ有効(紛失は自己負担)・値下げ絶対禁止・**第79回から赤字補填全面廃止**(準備金超過は自腹)・5月中に支出完了。準備金: 教室模擬2.5万(お化け屋敷は2クラス4万)/食品系教室模擬 食材5万+装飾1万/食品模擬8万(+機材別枠)/1年クラス4,000〜5,000円(資料間で差、要確認)。
- **決済**: au PAY のQRコードは翌年再利用(進路指導室保管)。手数料約2.6%を予算化。金券は当日・当該回のみ有効、未使用のみ払い戻し(100円単位)。
- **入場**: 正門一本化(北門・南門は退場専用)。再入場可・チケット非回収。チケットに生徒名・学籍番号記入で招待制を担保。チケット忘れは在籍生徒と連絡がつけば救済。
- **雨天判断**: 1日目=当日7:45最終判断 / 2日目=当日7:00。開会式=6:30判断(雨天は放送実施・集合教室表あり)、閉会式=13:30判断(体育館)。幕=朝7:00に運営委員長・実長・装飾担当で判断。野外はピロティ仮ステージ代替。
- **装飾規定**: テープは養生テープのみ、段ボール直書き禁止(紙を貼る)、壁ガムテ禁止、カーテン外さない。ポスター解禁は例年5/1前後(第79回資料では4/27、曜日表記ゆれあり)。
- **ゴミ**: 6分別(生ごみ/紙コップ類/可燃/缶・ペット/段ボール/不燃)。ゴミ本部=プール横。ステーションは第79回に3→5か所。管理委員(各クラス2名×1時間)。段ボールへ直接着色したものは回収不可。
- **安全・警備**: 警備本部=職員室(教頭)。消火器=各食品テント後ろ+校内各所。AED=職員室・体育教官室・神撫台グラウンド。シーバー(無線)第79回は13〜15台・ch1固定(資料間で台数差)。
- **撮影**: 校内祭=記録用1団体1名・事前申告。一般祭=Googleフォーム事前申請、生徒1名につき保護者1名(講堂は2階から)。
- **共用校地の制約**: 長田商業(平日活動時間・2階教室の使用可否が毎年の変動要因。2026年は「2F全教室前日準備不可」で決着)、青雲高校(文化祭翌日が日面=全施設使用禁止)。

## 8. 主要な反省・教訓(次回最優先で効く順)

1. **決済**: au PAY不満・PayPay要望が2年連続最多。割り勘不可・回線混雑・返金の課題も未解決。
2. **チケット**: 「家族が入れない」不満多数 vs 「客が少ない」の声も。家族枠/電子化/学年別配分/余剰再配分を早期議論。
3. **食品安全**: 生焼けが2年連続(罰則第3種)。調理チェック体制の制度化を。「疑い」でも重大。
4. **待ち時間表示の乖離**(20分表示→実際1時間超)は強いクレームに直結。支払列/受取列の全店統一ルールを。
5. **講堂の暑さ・キャパ超過**(体調不良者発生)。2部制・入替制の検討。野外×講堂の人気演目の時間被り。
6. **ゴミ箱の場所が分からない**が2年連続(分別運用自体は絶賛)。パンフ・地図に明記+増設。
7. **人員**: 総務は7人以上+経験者3人以上。装飾は1年生ゼロ問題。幕吊りは他パート応援10名前後が毎年必要。
